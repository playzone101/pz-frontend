## Getting Started

### Prerequisites

- must have Node.js (>= 18) installed

#### Installation

Clone the repository:

- `git clone https://github.com/your-repo/project-name.git`

- cd `project-name`

#### Install dependencies:

```bash
npm install
# or
yarn install
```

### Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
